About me

Soy un estudiante de diseño web con una fuerte pasión por la creación de sitios web bien estructurados y funcionales. Actualmente, me enfoco en el uso de HTML y CSS para desarrollar páginas web con una programación semántica que asegura accesibilidad y una experiencia de usuario intuitiva.

Mis proyectos actuales incluyen el diseño de una página web personal y el desarrollo de prototipos de interfaces para aplicaciones web. Mi objetivo es seguir mejorando mis habilidades y aplicar los principios de diseño web para crear soluciones digitales innovadoras y efectivas.
